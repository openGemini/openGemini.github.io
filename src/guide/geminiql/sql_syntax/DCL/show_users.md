# SHOW USERS

View all existing users and their administrator status

## Syntax

```sql
SHOW USERS
```

## Examples

```sql
>>> SHOW USERS
+-------+-------+--------+
| user  | admin | rwuser |
+-------+-------+--------+
| admin | admin | false  |
+-------+-------+--------+
3 columns, 1 rows in set
```
